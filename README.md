# Movie Tracker App

Movie Search Engine application - React + Vite

## Description

This project was created using React and JavaScript. Users are able to see current popular movies on the home page, and search for any movies using the search bars. They are then able to favorite the movies and track those of the "Favorites" page.

## Getting Started

### Dependencies

* react: ^19.2.0
* react-dom: ^19.2.0
* react-router-dom: 7.13.1

### Installing

* Download the files or clone the repository to a code editor

### Executing program

* Locate to the directory
```
cd frontend
```
* Install all dependencies
```
npm i
```
* Run the program
```
npm run dev
```

## Help
